# count(v1.1.0)
  A program to calculate data<br><br>
  We have extra system by module.

# Command
  We have several command to control data, you can find them by typing `key-word`.
  Command  | Use
  ---------|-------------
  `(v)add` | add data
  `(v)del` | delete data
  `extra`  | extra module
  `quit`   | quit
  `show`   | show data
