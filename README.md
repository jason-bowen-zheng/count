# count
  A program to calculate data<br>
  Version 2.1.0

# Command
  We have several command to control data, you can find them by typing `key`.
  * `(v)add`
    >Add data<br>
    *add KEY:VALUE*<br>
    *add KEY:VALUE1, VALUE2...*<br>
    *vadd VALUE*<br>
    *vadd VALUE1, VALUE2...*
  * `(v)del`
    >Del data<br>
    *del KEY*<br>
    *vdel [ALL]VALUE*
  * `info`
    >Show information of data<br>
    *info*
  * `key`
    >Show full key-word list<br>
    *key*
  * `quit`
    >Quit<br>
    *quit*
  * `show`
    >Show data<br>
    *show*
