# count
  A program to calculate data<br>
  Version 1.1.0

# Command
  We have several command to control data, you can find them by typing `key`.
  * `(v)add`
    >Add data
  * `(v)del`
    >Del data
  * `info`
    >Show information of data
  * `key`
    >Show full key-word list
  * `quit`
    >Quit
  * `show`
    >Show data
