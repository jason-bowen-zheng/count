# count
  A program to calculate data<br>
  Version 1.1.0

# Command
  We have several command to control data, you can find them by typing `key`.
  * `(v)add`
    >Add data
    *add KEY:VALUE*<br>
    *add KEY:VALUE1, VALUE2...*<br>
    *vadd VALUE*<br>
    *vadd VALUE1, VALUE2...*<br>
  * `(v)del`
    >Del data
    *del KEY*<br>
    *vdel [ALL]VALUE*
  * `info`
    >Show information of data
    *info*
  * `key`
    >Show full key-word list
    *key*
  * `quit`
    >Quit
    *quit*
  * `show`
    >Show data
    *show*
